Predicting the Success of NFL Quarterbacks

Logistic Regression, Random Forest, and Deep Neural Networks predict NFL quarterback success based on college metrics and pre-draft rankings. Comprehensive NFL/college dataset created with extensive data extraction, cleaning, and feature engineering.

Key Tools: Machine Learning (Logistic Regression, Random Forest, DNN with TensorFlow Keras), Data Extraction (requests, BeautifulSoup), Preprocessing (SimpleImputer, StandardScaler), Correlation Analysis, Feature Importance, K-Fold Cross-Validation, Hyperparameter Tuning (GridSearchCV).

SMS Spam Detection with TF-IDF & Support Vector Machine

Machine Learning pipeline using TF-IDF and SVC created to classify SMS messages as spam or non-spam, integrated with Gradio interface for real-time detection.

Key Tools: TF-IDF Vectorization, Support Vector Classifier (SVC), Gradio, Pipeline.

Predicting Employee Attrition with Multi-Branch Neural Network

Multi-branch Neural Network model applied and trained on employment data to identify factors driving employee attrition. Features two output layers for attrition prediction (binary) and department classification (multi-class).

Key Tools: Multi-Branch Deep Learning (DNN with TensorFlow Keras), One-Hot and Ordinal Encoding, Data Scaling, Feature Importance, Correlation Analysis, Hyperparameter Tuning, Cross-Validation, Evaluation Metrics.

Loan Repayment Prediction with Deep Learning

Deep Neural Network (TensorFlow Keras) trained and applied to predict loan repayment success, leveraging enhanced preprocessing and feature selection with Random Forest. Dataset includes academic, financial, and demographic data from previous loan recipients.

Key Tools: DNN with TensorFlow Keras, Scikit-learn (StandardScaler, classification report), RandomForestClassifier, Feature Importance.

Music Genres Forecasting with Prophet

Analyzed historical music genre trends from 1940 to 2019 to forecast future popularity shifts in genres 8 years into the future.

Key Tools: Prophet, Time Series Forecasting, Trend Analysis, Data Visualization (Seaborn, Matplotlib), Predictive Modeling, Data Cleaning.
